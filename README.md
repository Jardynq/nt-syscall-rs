This is an under development project that aims to allow inlining windows syscalls directly in code.
It also offsers some helpful shellcode features, such as heavens gate and cross architecture execution.

Aims to support:
Running x86 code on x86 host
Running x86 code on x64 host
Running x64 code on x64 host
Running x64 code on x86 host

Arm support might be considered at some point in the far future.
